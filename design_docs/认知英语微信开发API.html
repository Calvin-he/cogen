get /lessons
(lessonsGet)
Gets `lesson` objects

Return type

array[lesson]

Example data

Content-Type: application/json
[ {
  "num_of_viewer" : 123,
  "publish_time" : "2000-01-23T04:56:07.000+00:00",
  "num_of_comments" : 123,
  "name" : "aeiou",
  "description" : "aeiou",
  "id" : 123456789,
  "url" : "aeiou",
  "tags" : [ "aeiou" ]
} ]

Responses

200

Successful response
